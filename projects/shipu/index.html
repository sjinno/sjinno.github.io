Shípǔ

2021-03-18

Description

Shípǔ (meaning "recipes" in Mandarin Chinese) is an offline recipe maker by scraping the data of your favorite recipes from Cookpad JP and save it into its own html file. Useful if you have bad internet or are extremely tired of having to access to the website everytime you want to look up recipes.

Languages & tools: Rust and Python + HTML/CSS

Note: PyO3 is used in order to run Rust from Python.
